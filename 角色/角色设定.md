# 《万古邪帝》角色设定

---

## 主角：萧夜（邪帝）

### 角色卡

**姓名**：萧夜
**前世称号**：邪帝
**角色定位**：邪性智者，废材皮囊下的重生强者

**身份标签**：
- 前世：至尊巅峰强者，行事亦正亦邪，独行天下
- 今生：天风帝国京城第一废物纨绔，萧家落寞三少爷

**外貌特征**：
- 身形修长略显单薄，但站姿总是懒散中带一种说不出的锋锐
- 一双黑眸，平时半眯着像没睡醒，睁眼时却冷得刺骨
- 唇角常年挂着一抹似笑非笑的弧度
- 左手腕有一道细长的淡白旧疤（前世残魂印记）

**性格关键词**：
- 邪性——不按常理出牌，不吃道德绑架，喜欢反向设局
- 毒舌——插科打诨、调戏讥讽，把严肃场面搅成自己的节奏
- 护短——一旦划入"自己人"范围，无条件守护到底
- 底线硬——怕麻烦但不怕事，触底时出手极狠
- 话多但不啰嗦——平时喜欢调侃说笑，但真正发怒时话极少

**核心目标**：查清前世陨落的真相，逐一清算三圣地，重登巅峰

**核心动机**：
前世被最信任的人背叛，这笔账必须算清。不是为了仇恨，而是"我的命只有我能做主"

**致命弱点**：
- 对"自己人"过于信任，容易被人利用身边的人设局
- 骨子里仍有前世强者的傲，偶尔会因为不屑解释而吃亏
- 好奇心太重，遇到超出常理的事会忍不住探究

**口头禅/标志动作**：
- 标志动作：说话时喜欢用指节轻敲桌面/扶手，节奏越慢越危险
- 经典语气：轻笑一声，半眯着眼说"哦？"
- 开战前习惯：先歪头打量对方一会儿，然后叹口气

### 四层"邪"

| 维度 | 表现 |
|------|------|
| 行事邪 | 不按常规、不吃道德绑架、喜欢反向设局 |
| 语言邪 | 插科打诨、调戏讥讽、把严肃场面搅成自己的节奏 |
| 杀伐邪 | 一旦触底线，出手极狠，不留幻想 |
| 底线正 | 护亲、护兄弟、护故土，关键时刻能承担大义 |

### 情绪循环模式

搞笑铺垫 → 敌人挑衅/道德压迫 → 主角嬉笑应对 → 触及底线 → 气氛骤冷 → 血腥反杀 → 用亲情/兄弟/调侃收尾

## 主要配角

### 萧烈（祖父）

**角色功能**：家族旗帜 / 情感锚点 / 靠山
**核心特质**：护短暴躁老将军，嘴硬心软
**标志特征**：一身旧伤疤，吼声如雷但看见孙儿受伤手会抖
**与主角关系**：最疼萧夜但不知道怎么表达，只能用军棍和老铁拳
**关系弧线**：误解（废物孙子）→ 发现端倪 → 全力支持
**退场**：全书中期为守护家族牺牲，成为主角升级的核心动力

### 萧远（二叔）

**角色功能**：过渡家主 / 家族纽带
**核心特质**：平庸但忠心，夹在家族利益和皇室压力之间
**标志特征**：总是眉头紧锁，书房灯亮到深夜
**与主角关系**：长辈责任 > 真情，后期转化为真心扶持
**退场**：新家主上位后转入幕后

### 谢清霜（女主）

**角色功能**：情感线 / 战力成长线 / 身世线
**核心特质**：清冷高傲，天风帝国第一天才少女，实则身负诅咒
**标志特征**：一袭白衣，眉间一点朱砂，剑不离身
**与主角关系**：初期瞧不起（废材），中期怀疑（实力异常），后期并肩
**关键设定**：她体内封印着上古时期的某个存在，与九幽玄玉有共鸣
**退场**：无退场，贯穿全书

### 唐豆豆（兄弟/搞笑担当）

**角色功能**：最佳损友 / 信息源 / 笑点制造机
**核心特质**：胖、贪吃、怕死但对兄弟义气
**标志特征**：圆滚滚的身材却自称"玉树临风"，说话自带相声味
**与主角关系**：原主唯一真朋友，主角穿越后最先接纳的人
**家族背景**：唐家是帝国三大商会之一，他是最小最没出息的嫡子
**退场**：无退场

### 百里追命（初期小反派·进化型）

**角色功能**：小反派（1-5章障碍）→ 中后期成长型竞争对手
**核心特质**：嚣张、算计、但有自己的骄傲和底线
**初始动机**：欺压纨绔萧夜来博取名声和家族利益
**进化节点**：被主角击败后不服→正视差距→拼命追赶→从敌人变成对手
**退场**：后期成为亦敌亦友的存在

### 暗庭·影十三（幕后黑手线）

**角色功能**：暗杀者 → 线索提供者 → 更大阴谋的牵线人
**核心特质**：职业杀手，但有自己的行事准则
**层次**：初期执行者，中期引出暗庭，后期透露五年前"修炼意外"真相
**退场**：中后期为保护某个秘密而自我了断

## 反派层级

### 第一层：京城局（1-30章）

| 反派 | 身份 | 动机 | 手段 |
|------|------|------|------|
| 百里追命 | 百里家嫡子 | 踩萧夜上位 | 武力+权势 |
| 皇甫绝 | 皇甫家少家主 | 争夺资源 | 权谋+暗算 |
| 暗庭·影十三 | 杀手 | 执行命令 | 暗杀 |

### 第二层：四大家族（30-100章）

- 谢家（谢清霜的家族，内部派系）
- 百里家主
- 皇甫家主
- 皇室姜氏太子派系

### 第三层：风雪银城（类似异世邪君结构）

- 百年前与萧家有旧怨的上古势力分支
- 中期反派（约100-250章）

### 第四层：三圣地 + 背叛者（全书终极Boss）

- 前世围杀邪帝的三大势力
- 其中包含萧夜前世最信任的"兄弟"——真相贯穿全书
