# 第001章 醒来

"你、你没事吧？"

那张圆脸还在眼前晃。

萧夜躺在地上，花了三秒钟确认自己确实还活着——不，是换了个身体活着。

原主的记忆在脑海里翻涌，像是被人塞了一整座图书馆的信息。他一边过滤，一边打量眼前的胖子。

十六七岁，圆脸圆眼圆肚子，穿着一身价值不菲的锦袍，此刻急得满头大汗。

唐豆豆。

京城唐家商会最小、最没出息的嫡子。

也是原主唯一的朋友。

"三少爷你说句话啊！你可别吓我——"唐豆豆蹲下来，伸手想扶他又不敢碰，"我、我这就去叫人——"

"叫什么人。"

萧夜开口，声音沙哑，喉咙里还有血腥味。

他撑着地面坐起来，低头看了一眼身上的血迹。原主从醉仙楼三楼摔下来，后脑勺着地，按理说是死透了的。

换成原主，确实死透了。

唐豆豆见他坐起来，愣在原地。

"三、三少爷，你——"

"摔了一下，死不了。"

萧夜拍了拍身上的灰，目光不动声色地扫过醉仙楼的楼顶。

三楼临街的窗户还开着。

窗户后面，有一道人影。

还在。

凶手没有走。

他在确认萧夜是不是真的死了。

萧夜嘴角动了一下。

有点意思。

刚重生第一刻，就有人急着送线索。

他慢慢站起来，腿有点软——这具身体比他想像的还废。经脉枯涩，丹田干涸，肌肉松弛，一看就是好几年没正经修炼过。

但他毕竟是邪帝。

站还是要站住的。

"豆豆。"

"啊？"

"我刚才怎么掉下来的？"

唐豆豆挠了挠头："你、你喝多了，站窗户边上说你要飞——然后就——"

"就掉下来了？"

"就掉下来了。"

"我自己跳的？"

"你自己跳的。"

萧夜点点头。原主确实是喝醉了自己跳的——但那是被人灌醉之后。灌他酒的人，算准了他会站到窗边，算准了他会掉下去。

三楼不够高，摔不死一个曾经修炼过的人。

但原主的经脉被封了五年，身体比普通人还虚。

醉仙楼的地砖又是青石的。

算得真准。

萧夜抬头，又看了一眼三楼的窗户。

人影还在。

他笑了一下。

"豆豆，刚才谁跟我喝的酒？"

"百里家的管事，还有一个皇甫家的少爷，还有一个——"

"行了。"

萧夜拍了拍唐豆豆的肩膀。

"帮我传句话。"

"什么话？"

"就说萧家三少摔傻了，连自己叫什么都想不起来。"

唐豆豆瞪大眼睛："可你不是没——"

"照说。"

萧夜的语气很随意，但唐豆豆莫名觉得不能拒绝。

他点了点头。

萧夜转身往萧府的方向走，走了两步，又停下来。

他侧过头，声音不大不小，正好够三楼的人听见。

"对了。"

"刚才推我那个，穿灰衣服的，替我谢谢他。"

唐豆豆愣住了。

三楼的窗户，猛地关上了。

萧夜没有回头。

他继续往前走，嘴角的笑意慢慢冷下来。

刚才那句话，他故意说的。

推他的人明明穿的是黑衣。

他故意说错颜色。

凶手现在一定在想：怎么回事，他看到了？他没看到？他说的是不是我？

然后他会去质问那个"穿灰衣服"的人。

而穿灰衣服的那个，会以为凶手暴露了，会以为自己要被灭口了。

两个人会互咬。

萧夜不需要亲自审问。

他会坐在萧府里，等其中一个人送消息过来。

这就是他要的第一条线索。

回萧府的路上，天已经黑了。

萧夜一边走一边检查这具身体。越检查越头疼。

经脉枯涩得像是干涸了五年的河床。丹田里空空荡荡，别说玄气，连一丝灵气都感应不到。肌肉萎缩得厉害，刚才站起来的时候，腿都在抖。

唯一的好消息是骨骼还在。

而且——萧夜皱眉。

他隐约感觉到，这具身体的深处，藏着某种不一样的东西。

不是灵气，不是玄气，更像是一股被锁住的、极其暴烈的力量。

他试着用神识探查，刚触到那股力量的边缘，识海就是一疼。

像是被针扎了一下。

有意思。

这具废体里面，好像关着一头野兽。

萧夜没有继续探查。他现在太弱了，贸然去碰不该碰的东西，只会让自己再死一次。

慢慢来。

先弄清楚谁想杀他，然后找个安静的地方，把修为捡回来。

前世三百年的邪帝，就算从零开始，也不会比任何人慢。

回到萧府门口，他抬头看了一眼牌匾。

"萧府"两个字的金漆已经掉了大半，门口的石狮子缺了一只脚，门房的老头靠在椅子上打鼾。

曾经的天风帝国军方第一世家，如今只剩空壳。

他迈步进门。

门房老头醒过来，看了他一眼，又闭上了。

连个招呼都懒得打。

萧夜倒是不在意。他按照原主的记忆穿过前院，绕过正堂，走向自己住的那个偏僻小院。

萧府很大，但大半的院子都空着。家族的修炼资源早就被几个嫡系房头分干净了。原主这个曾经的废物少爷，分到的是整个府里最角落的地方。

院门虚掩着。

萧夜推开门，脚步顿住了。

院子里站着一个人。

身形修长，一身白衣，月光洒在她身上，像是给整个人镀了一层银。

她的脸很美，但冷得不像话。眉眼之间没有任何表情，就像一柄出鞘的剑，干净、锋利、拒人千里。

眉间一点朱砂，在月光下格外显眼。

谢清霜。

天风帝国第一天才少女，十六岁便已踏入玄师境。

也是原主做梦都不敢靠近的人。

萧夜靠在门框上，上下打量了她一眼。

"哟，稀客。"

谢清霜没有接话。她的目光从萧夜脸上扫过，像是在审视一件不太满意的商品。

"你没死。"

"让你失望了。"

"我没失望，"谢清霜的声音很淡，"我只是奇怪。"

"奇怪什么？"

"从三楼掉下来，后脑勺着地，就算不死也该昏迷三天。你居然能走回来。"

萧夜笑了。

"可能我头比较硬。"

谢清霜盯着他看了三秒钟。

"你不是萧夜。"

萧夜心里一紧，面上却不动声色。

"那我还能是谁？"

"我不知道，"谢清霜说，"但你绝对不是那个废物。"

她说完，转身就走。

走了两步，停下来。

"我今天来，是因为下午感应到你这里有一道不正常的玄气波动。"

"哦？"

"一道不属于任何活人的玄气波动。"

谢清霜没有回头。

"你最好别是什么不该出现在这里的东西。否则——"

她没有说完。

但萧夜听懂了。

他看着谢清霜的背影消失在月亮门后面，慢慢收敛了笑意。

她感应到了。

他重生时残存的九幽玄玉气息，被她捕捉到了。

天风帝国第一天才少女——不，这个女人不像表面上那么简单。

萧夜走进房间，关上门。

黑暗中，他低头看了一眼左手腕。

那里多了一道细长的淡白痕迹。

前世残魂的印记。

九幽玄玉果然跟着他来了。

只是现在，它还蛰伏在他身体里，需要足够的力量才能重新苏醒。

萧夜在黑暗中坐了很久，然后开口说了一句话。

"也好。"

"没有金手指的开局，才配得上邪帝的重生。"

窗外的月亮被云遮住了。

角落里，一双眼睛正在暗中注视着他的院子。

片刻后，那双眼睛消失了。

醉仙楼的某个密室里，有人低声说了一句话。

"他没死。"

"而且——他变了。"
