# 第三章 一句话

油灯的光先从楼梯口漫了下来。

然后是脚步声。一双厚底布靴踩在碎木片上，发出咔嚓咔嚓的脆响。萧夜在心里数着步子——七步，从楼梯口走到他刚才摔落的位置。这个人的步幅大约两尺半，体重不会超过一百五十斤，右腿比左腿稍微沉一点，应该是右撇子。

前世当杀手练出来的本事，换了一副壳子照样管用。

“在那边。”灰衣汉子举着油灯，光斑扫过翻倒的酒桌和砸烂的酒坛，最后停在碎木堆边缘。

“人呢？”尖嗓门从楼梯上半探着身子，不敢全下来。

灰衣汉子没有回答。他把油灯举高了一点，昏黄的光线一寸寸往碎木堆深处推进。

光停住了。

因为光照到了一双眼睛。

那双眼睛半眯着，懒洋洋的，像是刚从一场大梦里被人吵醒。眼睛的主人歪歪斜斜地靠在碎木堆上，头发上沾着碎木屑和酒渍，脸上白得没有一丝血色，嘴角却挂着一抹莫名其妙的微笑。

灰衣汉子的手抖了一下。

不是因为害怕——一个废物有什么好怕的？而是因为这跟他预想的不一样。他预想的是地上躺着一具尸体，或者顶多是一个摔断了脊椎、正在流血等死的半死人。他不应该看到一个正在笑的活人。

“你没死？”灰衣汉子的声音比他预想的更紧。

萧夜歪着头看着他。

这个歪头的角度是他精心计算过的——往左偏十五度，显得人很蠢，不构成任何威胁。与此同时，他的余光已经扫完了整个空间。灰衣汉子站的位置距离他约九尺，尖嗓门在楼梯口约十三尺，水桶应该还在楼梯中段，没有完全进入视野。

“我……”萧夜开口了，声音又轻又哑，“我是谁？”

灰衣汉子愣了一下，随即眼睛眯了起来。

“我是谁？我怎么会在这儿？”萧夜茫然地环顾四周，目光落在头顶那个贯穿三层楼板的大窟窿上时，露出了恰到好处的困惑，“那是什么？屋顶怎么有个洞？”

他演得很认真。认真的困惑，认真的恐惧，认真的虚弱。邪帝最擅长的事情之一就是演——不是因为他喜欢骗人，而是因为他发现，让人低估你是这世间最省力的杀人方式。

灰衣汉子盯着他看了很久。油灯在他手里微微晃动，投影在墙壁上的影子也跟着摇晃。

“萧三少爷。”他终于开口了，声音放得很慢，像是在试探一只受伤的猎物，“您不记得刚才发生了什么？”

“萧三少爷？”萧夜眨了眨眼，“我姓萧？”

灰衣汉子回头看了一眼楼梯口。尖嗓门冲他使了个眼色——那个眼色的意思很明显：既然摔傻了，那就好办了。

但灰衣汉子没有放松。他的直觉告诉他，有些东西不太对劲。一个从三层楼摔下来的人，就算不死也应该浑身是血、骨折筋断。可眼前这个废物除了脸色白了点、衣服乱了点，身上连个显眼的伤口都找不到。

这不正常。

“三少爷，”灰衣汉子蹲下身，把油灯放到地上，尽量让自己的声音听起来温和一些，“您再仔细想想，刚才是谁把您——”

“他。”

萧夜伸出一根手指，指向楼梯口。

尖嗓门的脸瞬间变成了死灰色。

“你放——”

“你闭嘴！”

灰衣汉子反手拔出腰间的短刀，动作快得几乎没有预兆。刀光在油灯下闪了一下，尖嗓门甚至来不及叫完那个字，刀就已经没入了他的胸口。

一刀毙命。干净利落。

水桶从楼梯中段冲了下来，看见尖嗓门瘫软在地的尸体，整个人僵住了。他张了张嘴，看看尸体，又看看灰衣汉子，再看看那个歪在碎木堆里一脸无辜的废物。

“为……为什么？”

“他是谁的人，你不知道？”灰衣汉子拔出短刀，在尖嗓门的衣服上擦干净血迹，语气平淡得像在说他今晚吃了什么，“百里少爷交代这件事的时候，只有你、我和麻杆在场。他不在场。他怎么知道要从三楼推的？”

水桶的嘴张得更大了。

萧夜在心里给灰衣汉子打了个好评。这个人执行力强、下手果断、杀完人还能冷静地给自己找理由——在前世，这种素质的人萧夜至少会给他一个副手的位置。只不过这一世，这个人是敌人。

这就比较遗憾了。

“萧三少爷，”灰衣汉子站起身来，把短刀收回腰间，脸上挤出一个恭谨的表情，“推您的人是那个尖嗓子的，他已经被我处理了。您安全了。我送您回府？”

萧夜茫然地点了点头。

水桶想去架萧夜，被灰衣汉子用眼神制止了。他还在试探——他让萧夜自己站起来，自己走路。如果一个从三层楼摔下来的人能自己走路，那就说明—

萧夜走了两步，然后以一个极其难看、极其逼真的姿势，左脚绊右脚，一头栽进了碎木堆里。

灰衣汉子最后一丝疑虑也消散了。

就这废物，别说伪装，就算让他光明正大地出手，也扛不住自己一巴掌。

“送三少爷回府。”他挥了挥手。

水桶汉子把萧夜架起来。萧夜的胳膊搭在水桶肩上，软绵绵的像条死鱼。他的头垂着，嘴角却在水桶看不见的角度微微翘了起来。

第一关，过了。

三条命里剩下两条。麻杆还在楼上守正门，水桶正在架着他往外走，灰衣汉子留在了后面拖尸体。萧夜的脑子转得比他脚下的步子快得多——

灰衣汉子至少玄士境。正面打，十个现在的他一起上也是送死。

但杀人，不一定要打架。

尤其是在他手里已经握住了第一条线索的时候。

刚才灰衣汉子那句话里，有一个关键信息：百里少爷交代这件事的时候，只有三个人在场。这意味着，真正下令推原主下楼的，不是灰衣汉子，不是水桶，不是尖嗓门。他们只是执行者。真正的指令来自楼上——来自那个"百里少爷"。

百里少爷是谁？

萧夜在原主残留的记忆碎片里翻了翻。百里是四大世家之一。百里家有嫡子和庶子若干，能被称为"百里少爷"的不止一个。但原主对这些人没有任何有用的记忆——他生前是个只知道喝酒赌钱的废物，连百里家有几个少爷都搞不清楚。

没关系。不知道就查。

水桶架着他走出了醉仙楼的后门。夜风扑面而来，带着帝都特有的煤烟和泔水混合的气味。萧夜借着月光看了一眼街道。

两排商铺，清一色的灰砖黑瓦。街道尽头是一座牌坊，牌坊上的匾额隐约可见三个字：朱雀街。

帝都的朱雀街。萧家侯府就在朱雀街尽头。

两条街。不到三百步。

这三百步，是邪帝重生之后走的第一段路。

水桶架着他走到萧府门口，把他往门柱上一靠，转身就走。萧夜靠在那只缺了耳朵的石狮子上，看着朱红大门上斑驳的漆色，看着门楣上那块褪了金的"萧府"匾额，看了很久。

这座府邸曾经很辉煌。

三代之前，曾祖萧天策以战功封侯，打的东玄域诸国闻风丧胆。萧家的铁骑一出，连皇帝都要起身相迎。那时候这座大门是帝都最气派的大门之一，朱漆鲜亮，铜钉铮亮，门前的石狮子连牙缝里都透着一股"谁也不服"的劲头。

现在呢？

石狮子缺了一只耳朵。门上的朱漆剥落得东一块西一块。连灯笼都比隔壁谢家少挂了两盏。

萧夜扶着门柱站起来。他没有敲门——因为他听到了门内的动静。有人在说话，声音压低但语气很急。

“张管家，三少爷到现在还没回来，要不要派人去找？”

“找什么找？他在外面喝死在外面最好。省得二爷还得替他操心。”

是张福。二房的管家。萧夜在脑海里搜出了这张脸——圆脸，永远挂着谄媚的笑，但笑不到眼睛里。

萧夜没有进去。

他靠在门柱上，闭上眼睛，把那半张当票从怀里摸出来，在指间翻来覆去地搓着。

当票上的字迹很潦草，只有两行。

抵押物品：玄铁令一枚。

抵押人：萧夜。

一枚玄铁令。原主那个废物，在坠楼之前，把一个玄铁令当了。

当铺的名字在当票背面：如意当铺，城南柳条巷。

萧夜把当票重新揣好。

他今天学到的事情很简单：有人要杀他，有人当了他的东西，而他连自己家的大门都不能光明正大地走进去。

但他不着急。

邪帝从来不着急。

他只是在等——等天亮，等身体恢复到能走路，等九幽玄玉苏醒，等那些藏在暗处的人一个一个地走到阳光底下。

然后一个一个地清算。

一声更夫的梆子从远处传来。

天快亮了。

萧夜抬起头，看着天边那一道若隐若现的鱼肚白，忽然轻轻地笑了一声。

上一世，他一个人从尸山血海里杀到绝天峰，连死都是孤零零的。

这一世，他有了一个家。

虽然破。

但是能修。
