# 《万古邪帝》项目概览

## 基本信息

- **书名**：万古邪帝
- **题材**：玄幻 / 重生逆袭
- **对标**：《异世邪君》（风凌天下）
- **定位**：男频爽文，邪性智者型主角，强者重生装废，认知差+护短+掀桌三爽叠加
- **目标平台**：起点中文网
- **风格定位**：高情绪、高口语、高反差。严肃时热血沸腾，轻松时插科打诨，开战时杀气极重，写亲情和兄弟时又突然拔高到很浓的抒情

## 核心卖点

1. **认知差爽**——强者灵魂藏在废物身体里，所有人都按旧情报判断他
2. **护短爽**——主角对自己人无条件守护，对敌人极狠
3. **掀桌爽**——主角先陪规则玩一会儿，然后在规则对自己人不利时直接掀桌
4. **情绪递进爽**——先铺委屈、误解、道德压迫，再让主角爆发

## 开局结构模型（异世邪君提取）

1. 前世巅峰身份 → 2. 意外死亡 + 神秘古物绑定 → 3. 异世废物身体 + 家族荣耀与衰败
4. 原主被暗害 → 5. 主角决定继续装废反向伪装 → 6. 第一波敌人按旧情报布局
7. 主角反向吞掉 → 8. 新关键人物章末登场

## 对标建议

从拆文库/异世邪君中提取以下模块：
- 文风：高情绪、感叹反问、排比热血、插科打诨、评书味旁白
- 节奏：循环"搞笑→挑衅→嬉笑应对→触底→骤冷→反杀→温情收尾"
- 人设：主角"邪"有四层——行事邪（不按规则）、语言邪（插科打诨）、杀伐邪（触底极狠）、底线正（护亲护兄弟）
- 金手指设计：玲珑小塔式神秘古物绑定，功能逐步解锁
