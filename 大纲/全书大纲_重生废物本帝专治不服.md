# 《重生废物？本帝专治不服》番茄平台全书大纲

> 书名：《重生废物？本帝专治不服》
> 题材标签：玄幻、重生、废材逆袭、扮猪吃虎、杀伐果断、护短、腹黑
> 平台定位：番茄（番茄小说）—— 节奏 > 代入感，强情绪 + 强噱头，爽点密度每2000字一个
> 总章节规划：350章（约70万字），分5卷

---

## 一、番茄平台全书定位

### 核心卖点（三句抓住读者）
1. **邪帝重生废物皮囊** —— 认知差爽感：读者知、敌人不知
2. **白天嗑瓜子遛鸟，夜里执掌生死** —— 角色反差 + 扮猪吃虎
3. **护短狂魔：动我的人，直接掀桌子** —— 情绪释放极致

### 目标读者画像
- 25-40岁男性，偏好爽文/逆袭/打脸
- 碎片化阅读，每章追更动力强
- 情绪需求：被压抑 → 释放 → 更压抑 → 更释放

### 番茄平台适配要点
| 维度 | 要求 | 本书实现 |
|------|------|---------|
| 开头策略 | 强情绪+强噱头，秒抓眼球 | 邪帝被背叛而死 → 重生废物身份 → 当场打脸 |
| 期待感 | 核心情绪深挖+反复提及 | "清算前世旧账"贯穿全书 |
| 爽点密度 | 每2000字一个微爽点 | 每3章（6000字）一个大爽点 |
| 节奏 | 快，不能停 | 每卷40-110章，无长铺垫 |
| 章节标题 | 噱头化，不干瘪 | 用每章最装X的一句话做标题 |

## 三、黄金三章设计（第1-3章）

### 第1章：他不是萧夜

**章节噱头标题**：剑穿胸口那一刻，他笑了一声

**核心结构**（约2000字）：
- 【起·300字】前世绝天峰战场闪回：邪帝萧夜被最信任的兄弟苏夜白一剑穿胸，三圣地围杀。临死前冷笑——他在锁魂阵里留了后手。
- 【承·500字】醒来：天风帝国萧家祖宅柴房。身体虚弱到站不稳，原主是帝都第一废物纨绔。迅速读取原主记忆碎片：七岁天才→九岁巅峰→五年前"修炼意外"→修为尽废。
- 【转·600字】家族困境切入：外面传来吵闹声。百里家二管家带人上门，说"萧家欠百里家的矿山该交割了"。祖父萧烈被气得咳血但无力反抗——萧家早已没落。
- 【合·600字】萧夜拨开人群走出去，歪着头打量百里家管事，半眯着眼笑。在场所有人脊背发凉——这个废物，怎么笑得不像是同一个人。萧夜轻飘飘一句："矿山？你跟本……跟我说规矩？"

**情绪曲线**：愤怒（背叛）→ 压抑（废物身体）→ 燃（走出去那一刻）

**本章完成指标**：
- [x] 人设基点：邪帝灵魂+废物皮囊，邪性智者
- [x] 切入点基点：前世背叛+今生困境双重压力
- [x] 金手指暗示：九幽玄玉在胸口微微发烫（不展开）
- [x] 期待感：他接下来要干什么？

---

### 第2章：你的规矩，我不认

**章节噱头标题**：他说矿山不交了，整个前厅没人敢出声

**核心结构**（约2000字）：
- 【起】萧夜当众撕毁矿山契约。百里家管事暴怒："你一个废物也敢——"萧夜抬头看他，只一眼，管事把后面的话咽回去了。
- 【承·金手指解锁】萧夜手指上翻出一枚黑色古玉——九幽玄玉。前世的本命物，随灵魂穿越而来。玄玉第一次觉醒：时间加速（10:1流速）+ 玄气提纯。他在众人眼皮底下，运转了一道《九幽碎虚录》的基础经脉循环。
- 【转】百里家管事恼羞成怒，命随从动手。萧夜轻描淡写躲过（前世邪帝的战斗本能还在），一掌拍在管事胸口——表面软绵绵，实则精准击中经脉死穴。管事当场瘫倒。全场死寂。
- 【合】萧烈震惊地看着孙子。萧夜转身回房时咳了口血（身体太弱强行运功的代价），但眼睛亮得吓人。当晚，他用九幽玄玉的时间加速，在柴房里进入了第一次修炼。

**情绪曲线**：紧张 → 爽（打脸）→ 期待（开始修炼）

**本章完成指标**：
- [x] 金手指基点：九幽玄玉解锁，明确时间加速+玄气提纯两大功能
- [x] 第一次爽点：一掌打瘫管事
- [x] 悬念钩子：身体太弱，需要抓紧时间变强

---

### 第3章：萧家，该站起来了

**章节噱头标题**：他只用了一夜，就从玄徒爬到了玄士

**核心结构**（约2000字）：
- 【起】一夜修炼（外界一夜=玄玉空间十天）。萧夜将修为从零恢复到玄士境。身体适应前世战斗本能，但锁天印（封住天弃邪脉的封印）还在，每突破一层都有走火入魔风险。
- 【承】清晨，萧烈在院中练枪。萧夜走过去，拿起一根木棍，指出萧烈枪法中的三处破绽。萧烈震惊——这是邪帝级的眼力。
- 【转·核心情感场景】萧烈沉默很久，问了一句："你……还是萧夜吗？"萧夜顿了片刻："我是萧夜。只是比从前多活了几年。"萧烈眼中含泪："不想说便不说，你是我孙子就够了。"
- 【合】萧夜在院中看着即将落败的萧家祖宅，对萧烈说："老头子，这帝都，该有人记得萧家姓什么了。"当天，他开始暗中布局，第一步——收回被百里家强占的三座矿山。

**情绪曲线**：燃 → 温情 → 期待（复仇正式开始）

**本章完成指标**：
- [x] 第一个目标建立：收回矿山
- [x] 情感锚点：爷孙关系确立，萧烈台词"你是我孙子就够了"
- [x] 期待感预埋：锁天印的秘密、苏夜白为什么背叛、三圣地是什么

## 四、第一卷：帝都风云（第1-40章）

### 卷定位
- **地图**：天风帝国帝都 + 周边矿区
- **核心冲突**：萧家复兴 vs 百里家/皇甫家的围杀
- **主角公开身份**：萧家三少爷（废物纨绔）
- **主角隐藏实力**：玄士→玄师→玄灵（卷末）

### 卷结构（三单元剧）

| 8-10章 | 百里家报复——雇佣暗庭杀手在路上伏击。唐豆豆（胖子损友）被绑做人质，宁死不跪。萧夜第一次真正动怒，血腥反杀全部杀手。唐豆豆从此死心塌地。百里追命被当众废掉右手 | 护短爽+友情锚 |
| 11-13章 | 百里家反扑——诬陷萧家私通敌国。皇甫家联合在朝堂上发难，萧家面临抄家危机 | 紧张压迫 |

**单元一终章状态总结**：
- 萧家地位：从没落→恢复军侯爵位

- 敌对状态：百里家重创但未灭，暗庭幕后老板未知
- 情感状态：谢清霜怀疑加深，唐豆豆坚定盟友
- 新悬念：影十三透露暗庭的雇主是"观星台"——一个不为人知的势力

**单元二终章状态总结**：

## 五、第二卷：青云之路（第41-90章）

### 卷定位
- **地图**：青云宗（东玄域第一宗门）+ 周边秘境
- **核心冲突**：宗门内部的资源争夺 + 锁天印真相
- **身份重置**：萧家三少爷→青云宗"关系户"新生（被所有人看不起）

### 卷结构（三单元剧）

#### 单元四：入宗立威（第41-55章）

| 章节区间 | 核心爽点 | 情绪目标 |
|---------|---------|---------|
| 41-43章 | 入学测试。萧夜被分配到最后一名的新生营。同营学员集体排挤——"关系户，滚出去"。萧夜搬了把椅子坐在训练场边上，慢悠悠嗑瓜子，看他们训练 | 日常装废+毒舌 |
| 44-46章 | 新生第一次试炼——进入妖兽森林。所有队伍故意甩开萧夜。结果他们在森林深处遭遇三阶妖兽团灭。萧夜一个人叼着瓜子走过来，轻描淡写解决妖兽。所有人从嘲笑→恐惧 | 碾压反转 |
| 47-49章 | 萧夜的实力引起宗门高层注意。外门长老质疑他的修为来源，要求检查经脉。萧夜露出被锁天印封住的废脉——"你自己看，我是废物。"长老查不出来，只能放人 | 隐藏实力+身份保护 |
| 50-52章 | 内门弟子百里追命出现（他被萧夜放走后，竟然也考入青云宗，想正面追赶）。两人在宗门竞技场上约战。萧夜只用一只手，百里追命输了，但不服输："下次我会赢。"萧夜第一次认真看他："我等着" | 亦敌亦友加深 |
| 53-55章 | 萧夜发现青云宗内部有一条完整的"养脉"产业链——宗门以培养天才为名，抽取弟子经脉中的特殊体质天赋，输送给上层势力。他的天弃邪脉如果是真的——就是整个系统最顶级的猎物。**谢清霜的神印也是养脉链条的目标** | 核心冲突揭示 |

#### 单元五：秘境真相（第56-75章）

| 章节区间 | 核心内容 | 情绪目标 |
|---------|---------|---------|
| 56-60章 | 青云宗秘境开启。萧夜和谢清霜组队进入。秘境中发现被"废弃"的弟子——经脉被抽干后封在秘境中自生自灭。萧夜第一次动了真正的杀心 | 愤怒燃+世界观黑暗面 |

## 十四、番茄平台特别注意事项

1. **章节标题**：禁用干瘪概括（"开始""觉醒""升级"），用每章最装X/最冲突的一句话
2. **开头秒抓**：第1章300字内主角登场，1000字内必须出爽点/期待点
3. **节奏控制**：大高控制5-10天内，小高控制2天左右，高潮后必须有日常过渡
4. **断章留钩子**：每章结尾必须有"接下来会怎样"的悬念
5. **情绪模块**：每段必须服务明确的情绪目标（愤怒/释放/紧张/更释放/余韵）
6. **广告分段**：番茄免费模式依赖广告，每2000字一个爽点保证用户愿意看广告
