# 《万古邪帝》全书大纲

> 目标平台：番茄男频
> 题材：玄幻重生、废材逆袭、扮猪吃虎、家族守护、热血复仇
> 预计篇幅：约 500 章，130 万—160 万字

## 一、核心定位

### 一句话梗概

一代邪帝萧夜遭挚友苏夜白背叛，被三圣地围杀，重生为帝都废物纨绔；他本想护住今生至亲、重登帝位清算旧账，却逐渐发现，这具“废体”来自敌人为夺取天玄界心培育了三百年的容器血脉。

### 核心卖点

1. **邪帝装废的认知差**：别人以为萧夜是任人欺辱的纨绔，他却用前世眼界、毒舌和反向设局掌控全场。
2. **护短型情感锚点**：前世孤身登顶，今生第一次拥有祖父、兄弟和爱人；“护住自己人”成为他比复仇更重要的成长。
3. **地图式升级复仇**：帝都、宗门、东域、银城、南海、西荒、北冥、三圣地、天门战场逐级扩大。
4. **同一谜底贯穿全书**：废脉、九幽玄玉、谢清霜的封印、前世围杀、三圣地异动，全部指向“天玄界心”。
5. **反派持续有效**：每卷反派都拥有独立目标，又是三圣地计划的一环；击败敌人既有即时爽感，也能推进终极复仇。

### 核心情绪

前期是“你们以为我废了”的认知差爽感；中期是“谁动我的人，我灭谁满门”的护短热血；后期是“天下都把我当邪魔，我偏要替天下斩神”的悲壮与痛快。

## 二、世界底层真相

### 1. 天玄界心

天玄大陆并非普通世界，而是一座残破的上古战界。世界深处孕育着“天玄界心”，掌控它便能调动一界法则，突破玄帝之上的天门境。

三千年前，界外的“天门神族”曾入侵天玄大陆。初代九幽邪帝以九幽玄玉封闭天门，将界心拆成九道本源，散入大陆九处地脉。

### 2. 三圣地的计划

北冥天域三圣地表面守护大陆，实际是天门神族留下的看门人。他们数百年来一直在寻找能够承载九道本源的肉身。

萧夜前世修炼九幽邪道，意外得到九幽玄玉，成为最接近界心的人。三圣地借苏夜白之手设局，在绝天峰围杀萧夜，夺走他的帝血与部分神魂。

但九幽玄玉在最后一刻带走萧夜主魂，使他重生到三圣地历经三百年培育出的“容器血脉”体内。

### 3. 今生废脉的真相

今生萧夜并非真正废物。他体内有一道“天弃邪脉”，能够吞纳九种互相冲突的本源。五年前，观星台发现邪脉即将觉醒，暗中用锁天印封住他的经脉，准备等容器成熟后献给三圣地。

原主的坠楼死亡，是观星台提前回收容器失败所致。前世萧夜的残魂恰在此时进入肉身，打乱了全部计划。

### 4. 谢清霜的封印

谢清霜是“天门之钥”的载体。她体内的太阴神印既能开启天门，也能反向封死天门。三圣地需要萧夜承载界心，需要谢清霜开启门户。

两人不是命定恋人，而是敌人计划中的两件祭品。他们的感情建立在共同挣脱命运之上。

### 5. 苏夜白的背叛

苏夜白是萧夜前世最信任的结义兄弟，也是三圣地圣子。

他接近萧夜最初就是任务，后来却产生了真感情。绝天峰前，他曾有机会警告萧夜，最终仍选择了圣地、长生和自己的野心。他在最后关头暗改了一道锁魂阵，使萧夜保住主魂，但这不是洗白，只是他无法彻底面对自己的背叛。

他的核心理念是：“牺牲一界众生，换少数人走出牢笼，总好过所有人困死。”

萧夜的回答是：“你想活，可以拿自己的命去争，没资格拿别人的命铺路。”

## 三、力量与金手指成长线

### 修炼九境

玄徒 → 玄士 → 玄师 → 玄灵 → 玄王 → 玄皇 → 玄尊 → 玄圣 → 玄帝

玄帝之上为终局境界：

- **界主境**：炼化一界本源，借世界之力而战。
- **天门境**：跨越世界壁垒，真正进入诸天。

### 九幽玄玉四阶段

1. **藏锋阶段**：时间减速、玄气提纯、功法推演，帮助萧夜重修根基。
2. **噬法阶段**：吞噬敌人残留的法则印记，解析弱点，但无法直接夺取完整修为。
3. **镇界阶段**：开启九幽界域，可镇压活物、保存地脉本源，也会放大萧夜心中杀念。
4. **归心阶段**：九道本源齐聚后，玄玉显露本体——天玄界心的外壳。萧夜必须在“炼化世界”与“归还众生”之间选择。

### 金手指限制

- 时间减速消耗神魂，不能无限闭关。
- 吞噬法则只提供理解，修为仍需萧夜亲自修炼。
- 每吸收一道本源，萧夜都会受到相应执念侵蚀。
- 九道本源集齐后，他可能失去人性，成为只维护世界运转的“界灵”。
- 谢清霜的太阴神印能帮他稳定神魂，但每次都要付出自身寿元。

## 四、人物长线

### 萧夜

**起点**：把今生当作一次复仇工具，不愿真正接纳任何人。
**中段**：因萧家、唐豆豆、谢清霜而重新拥有牵挂，也因此第一次真正害怕失去。
**低谷**：萧烈为护萧家战死，萧夜险些被九幽杀念吞噬，决定以屠尽仇敌来逃避悲痛。
**终点**：拒绝成为高高在上的界灵，把界心归还天地；他不再以孤独证明强大，而是带着活下来的人共同守住世界。

### 谢清霜

**起点**：冷傲克制，把所有感情视为弱点。
**中段**：从怀疑萧夜，到与他共享秘密，再到主动承担封印反噬。
**冲突**：她发现自己只要活着，三圣地就能以她为钥开启天门，因而决定自毁神印。
**终点**：萧夜没有替她做选择，两人共同将“钥匙”改造成“门闩”；她从祭品变成决定世界命运的人。

### 唐豆豆

**起点**：怕死、贪财、被家族视为没用的纨绔。
**成长**：利用商会天赋建立横跨大陆的情报与运输网，成为萧夜最可靠的后勤核心。
**关键高潮**：西荒大战中，他明知城门一开必死，仍亲自押送军资穿过死地。
**终点**：执掌万宝盟，却依旧是那个敢对邪帝讨债的兄弟。

### 萧烈

**功能**：萧夜今生最重要的亲情锚点。
**弧线**：从担忧孙儿沉沦，到知道萧夜身藏秘密却选择不问，再到把萧家未来交给他。
**退场**：第四卷末，为保护风雪银城百姓和萧家后辈，独守最后一道城门战死。其死不是给主角送怒气，而是完成“将军守土”的人物理想。

### 苏夜白

**起点印象**：彻底背叛萧夜的前世仇人。
**中段揭示**：他既是三圣地圣子，也是天门计划的主持者；他曾救下萧夜一缕主魂。
**人物本质**：珍惜萧夜，却更爱自己的道路；有愧，但从不愿放弃既得利益。
**终局**：在天门前试图再次说服萧夜合作，被萧夜正面击败。萧夜承认两人曾是兄弟，也亲手为这段兄弟情送葬。

# 第一卷：帝都藏锋（第 1—55 章）

### 大纲四检

- **目标情绪**：废材逆袭、扮猪吃虎、护短反杀。
- **核心冲突**：萧家军权被夺，四大家族准备借“通敌案”灭萧家。
- **节奏**：前十章高速建立人设；中段连环破局；后段帝都政变集中爆发。
- **伏笔**：原主坠楼、锁天印、观星台、谢清霜神印、萧家祖令。

### 起始状态 → 结束状态

京城第一废物、萧家待宰羔羊 → 帝都无人敢轻视的萧三公子、萧家重掌军权；无修为 → 玄师初境。

## 单元一：邪帝醒来（第 1—10 章）

1. 绝天峰上，苏夜白引三圣地围杀萧夜。萧夜自爆帝域，九幽玄玉带走主魂。
2. 萧夜在醉仙楼坠楼后的废物少爷体内醒来，发现凶手仍在楼上确认他的死活。
3. 他装作失忆纨绔，反用一句话让行凶者误杀同伙，带着第一条线索回府。
4. 萧家族老逼萧远交出主脉产业，萧夜当众胡闹，实则让内鬼主动暴露账目漏洞。
5. 百里追命设宴羞辱，萧夜不动武，仅凭眼力点破其功法死穴。
6. 当夜九幽玄玉苏醒，器灵九幽现身，确认萧夜体内不是废脉而是被封住的天弃邪脉。
7. 萧夜以《九幽炼气诀》重入玄徒，却发现每次冲击封印都会惊动某个远方阵眼。
8. 唐豆豆带来坠楼前的赌债册，萧夜从伪造笔迹中锁定百里家管事。
9. 谢清霜为调查异常玄气接近萧夜，被他故意调戏掩盖修为，双方留下极差的第一印象。
10. 百里追命在武市逼唐豆豆下跪，萧夜第一次公开出手，一指废掉其最自傲的右手。

**小高潮**：全城第一次意识到，萧家废物可能一直在装。
**章尾大钩子**：观星台星盘上，代表“容器”的死星重新亮起。

## 单元二：萧家有鬼（第 11—21 章）

- 百里家以萧夜伤人为由逼萧家交出三座矿山，皇甫家同时切断军粮。
- 萧夜主动接管最烂的城南商铺，用唐豆豆的渠道把废矿炼渣变成军用玄铁，完成第一次点石成金。
- 谢清霜暗查萧夜，发现他每次胡闹都恰好让敌人损失最大，却始终抓不到证据。
- 暗庭杀手影十三夜袭。萧夜故意放他走，在其身上留下只有雇主才能触发的血印。
- 血印指向萧家二房。萧夜没有立刻杀内鬼，而是借他向百里家送出假军报。
- 假军报引出边军粮饷案，百里家私仓暴露，皇室被迫公开查案。
- 萧夜在查账时发现原主坠楼前曾试图把一枚祖令交给萧烈。
- 谢清霜体内神印第一次与九幽玄玉共鸣，两人同时看到一扇染血天门的幻象。
- 二房劫持萧家幼子逼萧夜退让。萧夜单人赴约，救人后血洗据点。
- 萧夜将内鬼罪证交给萧烈，却隐去封印与重生秘密。萧烈只说：“不想说便不说，你是我孙子就够了。”

**收益结算**：萧家夺回矿山；唐豆豆获得独立商队；萧夜突破玄士。
**新开环**：假军报中有一枚观星台密印，说明百里家也只是棋子。

## 单元三：边关死局（第 22—33 章）

- 北境传来萧家军大败消息，萧烈被诬陷通敌，皇帝命其回京受审。
- 萧夜判断所谓败报是诱使萧烈离开边军的陷阱，主动请命前往雁回关送旨。
- 路上遭三拨截杀，他反向利用暗庭杀手清除皇甫家死士，并收服影十三。
- 雁回关粮草被烧、蛮族压境、主将中毒，萧夜在一夜内稳住军心。
- 他用前世阵道改造残破城防，以三千残军制造十万伏兵假象。
- 蛮族退兵是假，真正目标是地下的第一道界心阵眼。
- 谢清霜奉太后密令赶来，与萧夜首次并肩。两人一冷一邪，配合仍处处冲突。
- 萧夜在阵眼中发现锁天印与自己经脉封印同源，确认敌人早在五年前就盯上原主。
- 镇北王姜玄现身，承认通敌案是他推动，却只知道圣地使者许诺让他登基。
- 萧夜借蛮族王庭内乱反杀姜玄先锋，再将真实战报送回帝都。
- 萧烈率军回援，祖孙联手守住雁回关，姜玄被迫提前发动帝都政变。

**大爽点**：所有人等萧夜送死，他却以废物之名挽救一关。
**卷中转折**：真正战场从边关转回帝都，萧家满门已被控制。

## 单元四：帝都掀桌（第 34—45 章）

- 萧夜轻装潜回帝都，发现萧府被禁军包围，唐豆豆和谢清霜分别掌握商路与宫中线索。
- 姜玄、百里家、皇甫家准备在太后寿宴上公开定萧家通敌之罪。
- 萧夜故意让影十三“刺杀”自己，把暗庭雇佣契约送进大理寺。
- 唐豆豆抬高全城粮价，逼隐藏军粮仓主动开仓，锁定叛军部署。
- 谢清霜公开与家族决裂，把谢家掌握的祭阵名单交给萧夜。
- 寿宴之上，百里家先呈伪证，姜玄逼皇帝下旨灭萧家。
- 萧夜一份份拆穿伪证，再让所谓“战死”的边军证人入殿。
- 姜玄发动伏兵，帝都九门同时关闭；萧夜则提前换掉九门军械。
- 萧烈率边军压城却不入城，避免坐实谋反；萧夜在宫内解决政变，祖孙形成内外夹击。
- 镇北王强行开启皇宫下的界心阵眼，试图用全城百姓血祭换取力量。
- 萧夜解开第二层邪脉封印，以玄士之身越境斩玄师，玄师初境成。
- 姜玄临死前说出“青云宗会来收走成熟的容器”。

## 单元五：藏锋出京（第 46—55 章）

- 百里、皇甫两家倒台，萧家洗清冤屈，萧烈重掌北军。
- 皇帝要封萧夜为侯，将他留在帝都充当制衡萧家的质子。
- 萧夜当众拒封，只要一张青云宗入门帖，主动把自己送出皇权棋盘。

# 第三卷：东域称王（第 111—170 章）

#### 单元五：东域新王（第 160—170 章）

- 萧夜突破玄王巅峰，邪盟成为东域第一势力。
- 风雪银城送来一口冰棺，棺中是萧家失踪百年的先祖。

**卷末状态**：萧夜从个人强者成长为东域领袖。
**下一卷钩子**：冰棺先祖睁眼，只说了一句：“萧家血脉，是寒月祭坛最后一把锁。”

## 七、爽点与节奏控制

### 每卷固定节奏

1. 前 3 章给出新地图压迫与明确目标。
2. 每 3—5 章闭环一个小冲突。
3. 每 10—12 章完成一个剧情单元和中型爽点。
4. 每卷 60% 左右发生第一次格局反转。
5. 每卷最后 12—15 章进入连续高潮，不在高潮中插无关支线。
6. 卷末完成“敌人被清算、资源有增值、关系有变化、新危机已开启”四项结算。

### 核心爽点轮换

- 第一卷：智谋破局、公开打脸。
- 第二卷：宗门规则反杀、底层逆袭。
- 第三卷：战场统帅、势力建立。
- 第四卷：血脉碾压、护族复仇。
- 第五卷：秘境探索、越境斩圣。
- 第六卷：守城群像、众生拥戴。
- 第七卷：天骄争霸、斩碎前世。
- 第八卷：旧仇清算、圣地覆灭。
- 第九卷：众生共战、斩神断门。

### 必守原则

- 萧夜可以吃亏，但不能因降智导致全局崩盘。
- 下行情节由敌人实力、世界规则或守护代价造成，不能靠主角反复犯错。
- 每获得一道本源，必须立刻解决一个具体危机，同时带来一种新代价。
- 苏夜白每次出场都要推进“旧情、背叛、理念”至少一项，不能只做谜语人。
- 谢清霜拥有独立决策和战斗成果，不能长期充当被救目标。
- 唐豆豆的商业与情报线必须真实改变战争结果，不做纯搞笑挂件。
- 萧烈之死前至少完成两次理念传承，死亡后仍通过军规、旧部和萧夜选择产生影响。
- 三圣地覆灭不是全书终点，而是复仇主线闭环；终局回答“强者是否有权替众生选择”。

## 八、结局落点

萧夜前世追求的是“没人能决定我的命”，今生最初仍只想夺回自己的选择权。

走到最后，他才明白真正的自由不是自己站到最高处，而是不允许任何人——三圣地、天门神族，甚至他自己——替所有人决定命运。

所以他最终没有炼化天玄大陆，没有建立第四圣地，也没有成为永恒界灵。

他仍是邪帝。

邪的是不认天命、不跪权威、不接受用冠冕堂皇的理由牺牲无辜。

帝的是能执掌最强力量，也能在拥有整个世界时把它还给众生。
